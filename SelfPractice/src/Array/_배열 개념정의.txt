배열(Array): 공통 이름으로 참조되는 자료형의 모음.
배열 선언 시 배열의 크기를 지정해야 하므로 *정적 데이터 구조*라 부름.

1. 배열의 개념: 같은 유형의 데이터를 모은 집합(모음).
인덱스(index) 0에서 시작.

6.2 1차원 배열과 2차원 배열 
1. 1차원 배열 선언: 단일 인덱스를 사용하여 요소를 저장.
int[] myArr = new int[5];
배열 선언		  배열 생성(5개의 정수를 저장하는 배열)

new 키워드를 사용하여 메모리 공간에 값을 저장.
int[] myArr = new int[5];
myArr[0] = 10;
myArr[1] = 10;
myArr[2] = 10;
myArr[3] = 10;
myArr[4] = 10;

2. 2차원 배열 선언: 데이터가 행과 열에 저장되는 표 형식으로 나타남.
int[][] myArr = new int[3][4];
